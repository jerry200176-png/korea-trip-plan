# Pre-departure checklist

Use with [`data/timeline.yaml`](../data/timeline.yaml). Check items only when actually done.

## Documents

- [ ] Passports valid (verify airline + Korea entry rules)
- [ ] K-ETA / visa status confirmed from **official** site (not blogs)
- [ ] Travel insurance policy PDF offline on both phones
- [ ] Emergency pack PDF offline
- [ ] Handbook site added to home screen / PDF saved

## Money & connectivity

- [ ] Card overseas usage enabled (check **your bank’s** official instructions)
- [ ] Know how to decline DCC (dynamic currency conversion)
- [ ] Small KRW cash plan
- [ ] eSIM / SIM / Wi-Fi decided and installed instructions saved

## Bookings (store codes offline — never commit)

- [ ] Flights
- [ ] Seoul lodging
- [ ] Busan lodging (if dual-city)
- [ ] KTX / rail (if needed)
- [ ] Experiences that require reservation

## Health & comfort

- [ ] Comfortable walking shoes broken in
- [ ] Blister care / meds
- [ ] Instant camera film + spare battery plan

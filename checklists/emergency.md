# Emergency checklist

## Immediate numbers (Korea)

| Service | Number | Notes |
|---------|--------|-------|
| Police | 112 | |
| Fire / ambulance | 119 | |
| Taipei Mission in Korea | See official site | [roc-taiwan.org/kr](https://www.roc-taiwan.org/kr) — re-verify phones before trip |

## Do this

1. Get to a safe place; call 119/112 if needed.  
2. Contact insurance emergency line (number stored **offline**, not in git).  
3. Contact Taipei Mission if passport lost or serious incident.  
4. Use Emergency Pack PDF for lodging address in Korean + key phrases.  
5. Message each other with location pin (Naver/Kakao/Apple).

## Lost passport

- Police report + Mission assistance — follow official Mission instructions (re-check URL before trip).

## Medical

- 119; hotel desk can help call; carry allergy phrases from handbook.

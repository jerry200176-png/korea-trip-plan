# 韓文實用句

短句優先，現場可指著螢幕。

## 飲食需求

| 中文 | 韓文 |
|------|------|
| 我們不喝酒 | 저희는 술을 마시지 않아요 |
| 可以不要甲殼類海鮮嗎？（蝦蟹等） | 갑각류는 빼 주실 수 있나요? |
| 我對蝦過敏（若適用・請自行確認） | 새우 알레르기가 있어요 |
| 豬肉湯飯請給我一份 | 돼지국밥 하나 주세요 |
| 不要太辣／可以辣 | 덜 맵게 해 주세요 / 맵게 해 주세요 |
| 謝謝 | 감사합니다 |

## 求助

| 中文 | 韓文 |
|------|------|
| 幫得我忙嗎？ | 도와주세요 |
| 我想去這裡（指地圖） | 여기로 가고 싶어요 |
| 請叫救護車 | 구급차를 불러 주세요 |
| 我迷路了 | 길을 잃었어요 |
| 會說英語嗎？ | 영어 하세요? |

## 交通

| 中文 | 韓文 |
|------|------|
| 請到這個地址 | 이 주소로 가 주세요 |
| 請在這裡停車 | 여기 세워 주세요 |

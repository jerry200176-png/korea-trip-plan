## Hanbok candidates (minimum 3)

> 店家細節以預約頁／當日店面為準。下列為 **可追蹤候選**，不是已付款鎖定。

### H1 — Hanboknam（한복남）景福宮站周邊

| Field | Evidence / status |
|-------|-------------------|
| 位置 | 景福宮站周邊（多來源指 Exit 2/3 一帶） |
| 語言 | 旅遊平台／導覽文常見英文流程；**現場再確認** |
| 押金 | Visit Seoul 文章級說明常見押金或護照抵押（例：₩50,000 或護照）— **各店不同，以店家為準** |
| 髮型 | 多店含簡單編髮／丸子頭選項 |
| 時長／價格 | 常見 2–4 小時或全日；約 ₩10,000–₩40,000 帶（平台區間） |
| 雨天 | 宮殿戶外為主 → 雨備改室內韓屋咖啡／縮短 |
| 歸還 | 逾時可能扣押金；預留 30 分緩衝 |
| 適合 | 第一次出國、要近宮殿 |
| Verdict | **Primary shortlist** |

## Area compare after palace

| Area | Value | Cost | Verdict |
|------|-------|------|---------|
| **北村** | 韓屋屋頂／巷弄合照最強 | 坡、居民區需安靜、人潮 | **Retain Core photo walk（可縮短）** |
| **仁寺洞** | 文創街、茶館、購物輕 | 與宮殿同圈可走 | **Optional 補段** |
| **益善洞** | 咖啡／復古拍照 | 另段步行；易變打卡擁擠 | **Optional／低體力可跳過** |

**結論：** Core = 韓服 → 景福宮 → 北村短走 → 休息。仁寺洞／益善洞不預設全塞。

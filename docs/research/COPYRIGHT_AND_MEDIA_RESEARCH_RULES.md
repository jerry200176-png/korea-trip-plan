# Copyright and Media Research Rules

**Updated:** 2026-07-20  
**Related:** [`docs/MEDIA_POLICY.md`](../MEDIA_POLICY.md), [`data/visual-candidates.yaml`](../../data/visual-candidates.yaml)

## Research text

Allowed:

- Short factual paraphrase with URL citation  
- Recording hours/prices only when Tier A confirms  
- Linking to creator content for discovery  

Forbidden:

- Copying long articles, full blog posts, or YouTube transcripts  
- Pasting Instagram / Threads post bodies  
- Quoting enough of a guide that the product becomes a substitute for the original  

## Media assets

### Round 1

- Catalog existing media and gaps only  
- Record visual *candidates* in `data/visual-candidates.yaml`  
- **Do not** download 40–70 new images in this round  

### License path before any download (later rounds)

| Source class | Rule |
|--------------|------|
| Original AI illustration (project-owned prompts) | Allowed as illustrative only; never as place evidence |
| Unsplash / clearly licensed stock | Keep license + creator + source URL; no faces if avoidable |
| KOGL / official tourism | Jerry must confirm commercial/modification rights before apply |
| Creator photos (IG/YouTube stills) | Default **no** without explicit permission |
| Screenshots of apps / maps | Prefer redrawn diagrams; avoid trademark-heavy UI |

### Visual function tags (required)

Every kept or candidate image must declare one primary function:

`Orient` · `Explain` · `Compare` · `Inspire` · `Identify` · `Warn` · `Remember`

Decorative-only images are candidates for replace/remove.

## Factual vs illustrative

| Class | Can support operational claims? |
|-------|----------------------------------|
| Factual photo of named place (licensed) | Soft support only; hours still need Tier A |
| AI illustration | Atmosphere only |
| Diagram / map redraw | Teaching OK if data sources cited |

## PII

No traveler surnames, faces of strangers without rights, passport/boarding docs, or lodging booking screenshots in public outputs.

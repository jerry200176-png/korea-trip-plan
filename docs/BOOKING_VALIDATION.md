## D3 ticket verification (Provisional D3-B)

Compare **two-person total trip cost**, not fare alone.

### Priority filters

1. **Times & total movement**
   - Outbound: arrive ICN **no later than 18:00** (avoid late-night Day 1 haul).
   - Return: PUS depart **not before 13:00**, prefer **14:00–18:00** (protect Day 6 / Day 7).
   - Prefer direct; do not accept long connections to save a small fare.
2. **True all-in price (two people)**
   - Include: tickets, checked bags, seat fees, card FX, extra KTX, airport transfers.
   - Decision threshold: if D3-B all-in is **≤ NT$5,000 more** than D3-A, prefer **D3-B** (saves return KTX energy / half-day risk).
3. **Change/cancel reliability**
   - Compare with **20–23 kg** checked bag baseline per person.
   - Prefer clear change/cancel rules; if split tickets, check **both** legs.

### Plan B (D3-A)

Only if D3-B fails moment/price/reliability:

- TPE ↔ ICN + KTX round-trip Busan
- Hard gates: ICN return on 3/27 preferably **after 20:00**; planned KTX arrive Seoul Station **≥ 5 hours** before takeoff
- If gates fail → D3-C (re-open dates/routing) — do not sacrifice safety buffer

### Compare set

- D3-B: at least **3** option sets  
- D3-A: at least **2** option sets  
- Each row: all-in price, times, change/cancel, total movement

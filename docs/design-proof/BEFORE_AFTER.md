# Before / After — Visual design

## Before (pre–Warm Editorial)

- Dark / system dual theme dashboard (`--bg: #0f1419`).
- Home labeled **Dashboard** with readiness gates, Founder jargon.
- PDFKit raw English/ASCII-leaning text dump.
- Card-panel engineering shell; no editorial serif journey framing.

## After (Our First Korea — Warm Editorial Journey)

- Light warm rice canvas (`--color-canvas: #f7f4ed`), forest green primary, warm orange accents.
- Home is a couple travel book: nights per city, quick-find chips, decision statuses in Chinese.
- Day pages use route-line timelines; Today Mode prioritizes one thing + map + Plan B.
- PDF via HTML + Chromium print with Noto CJK, cover/TOC/day spreads.
